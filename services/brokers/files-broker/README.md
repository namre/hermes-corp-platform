# files-broker — файлы / календарь / почта

**Заглушка** хранит файлы в памяти и отдаёт фейковые ссылки — чтобы топология
поднималась без секретов. **Образец** — универсальный блок на базе Yandex 360
(disk, calendar, mail, telemost, wiki) с per-user OAuth и шифрованием токенов.

- Каталог инструментов и что прописать в конфиге — [TOOLS.md](TOOLS.md)
- Инструкция для LLM-ассистента собрать ваш вариант — [IMPLEMENT.md](IMPLEMENT.md)

Заменяемо на Google Workspace / Microsoft 365 / S3+CalDAV без правки остальной
платформы — контракт тот же.
