# llm-gateway — единая точка к моделям (образец: LiteLLM)

Агент ходит к моделям через один OpenAI-совместимый шлюз. Это даёт смену
провайдера/модели без правки агентов, виртуальные ключи на сервис, лимиты и фолбэк.

- Конфиг-образец: [config.example.yaml](config.example.yaml)
- Генератор виртуальных ключей (без секретов): [gen_key.sh](gen_key.sh)
- Инструкция для LLM-ассистента: [IMPLEMENT.md](IMPLEMENT.md)

Универсальный блок: LiteLLM можно заменить любым OpenAI-совместимым прокси.
Платформа знает только `LLM_BASE_URL` и виртуальный `LLM_API_KEY` (см. `.env`).
