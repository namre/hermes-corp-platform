# identity-proxy

Единая точка входа. Резолвит личность через `corp-dir-broker`, поднимает
персональный runtime через `provisioner`, прогоняет текст через детектор
подозрительных паттернов и форвардит запрос в runtime пользователя.

Скелет: резолв и роутинг готовы; включите форвард (отмечен `TODO(REPLACE)`)
под ваш транспорт к Hermes-Agent.
